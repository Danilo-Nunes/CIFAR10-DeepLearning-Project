# CIFAR10-DeepLearning-Project
Este projeto tem como objetivo desenvolver um modelo de aprendizado de maquina deep learning e explorar diferentes arquiteturas de redes neurais convolucionais (CNNs) para reconhecimento de padrões em imagens, garantindo alta taxa de acerto mesmo diante de ruídos, distorções e caracteres sobrepostos — características comuns em CAPTCHAs reais.
